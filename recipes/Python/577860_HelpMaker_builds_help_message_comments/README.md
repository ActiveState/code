## HelpMaker: builds help message from comments in the code 
Originally published: 2011-08-28 09:59:57 
Last updated: 2011-08-28 14:34:58 
Author: Yang  
 
This tiny module is to make your comments in the source code a bit more useful, to be displayed as help messages. With the HelpMaker, once you finish writing/updating the comments, the help messages are ready.\nOne more reason to work on your comments =)
