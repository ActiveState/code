## Simple PDF Bookmark / Table of Contents Maintenance

Originally published: 2016-06-20 18:14:38
Last updated: 2016-06-20 18:14:38
Author: Jorj X. McKie

PyMuPDF now supports a simple interface for maintaining table of contents / bookmarks and metadata for PDF documents.\nBy manipulating ordinary, elementary lists and dictionaries you can add, delete, modify a PDF's metadata and table of contents.
