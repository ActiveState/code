## TRAC Interpreter -- Sixties programming languageOriginally published: 2010-08-22 19:05:09 
Last updated: 2010-08-22 19:05:09 
Author: Jack Trainor 
 
A Python implementation of an interpreter for the TRAC programming language developed by Calvin Mooers in the early 1960s. This implementation reconstructs Mooers' original algorithm in Python and supports only a limited number of TRAC primitives for demonstration purposes.\n
