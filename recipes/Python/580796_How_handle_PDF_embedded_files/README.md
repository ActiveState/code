## How to handle PDF embedded files with PyMuPDFOriginally published: 2017-05-04 15:54:22 
Last updated: 2017-05-04 15:54:58 
Author: Jorj X. McKie 
 
Version 1.11.0 (based on MuPDF v1.11) allows exporting, importing and interrogating files embedded in a PDF.\n\nPDF "/EmbeddedFiles" are similar to ZIP archives (or the Microsoft OLE technique), allowing arbitrary data to be incorporated in a PDF and benefit from its unique features.
