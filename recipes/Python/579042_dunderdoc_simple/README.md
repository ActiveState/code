## dunderdoc, a simple Python introspection tool

Originally published: 2015-03-30 20:22:19
Last updated: 2015-03-30 20:22:20
Author: Vasudev Ram

The dunderdoc() function, in module dunderdoc.py, is a simple Python introspection tool that I wrote. It prints the __doc__ attribute (the docstring) of each item in a list of names given as argument to it. \n\n
