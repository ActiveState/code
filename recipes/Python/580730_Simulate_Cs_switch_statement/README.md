## Simulate C's switch statementOriginally published: 2016-12-11 16:28:49 
Last updated: 2016-12-11 16:28:50 
Author: Vasudev Ram 
 
This recipe shows a Python construct that can behave somewhat like C's switch statement. It is not a perfect one-to-one simulation. But it does have some of the features of the C switch. One feature not supported is the fall-through feature in C's switch.\n\n
