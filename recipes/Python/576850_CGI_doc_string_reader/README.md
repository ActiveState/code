## CGI doc string reader 
Originally published: 2009-07-20 07:09:51 
Last updated: 2010-04-13 14:06:06 
Author: Timothy Makobu 
 
Displays docstrings of attributes of the given Python module on a web browser.\nDrop it into the cgi-bin of your web-server, and access it like this, \nfor example:\n\nhttp://localhost/cgi-bin/cgidir.py?m=socket\n
