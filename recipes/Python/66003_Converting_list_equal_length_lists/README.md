## Converting a list of equal length lists into XMLOriginally published: 2001-07-08 03:55:00 
Last updated: 2002-07-21 11:35:13 
Author: Julius Welby 
 
This module takes a list of equal length lists and converts it into XML.\n\nIf the first sublist is a list of headings, these are used to form the element names of the rest of the data, or these can be defined in the function call. Root and "row" elements can be named if required.
