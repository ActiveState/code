## Convert a youtube transcript in srt subtitleOriginally published: 2010-11-12 01:59:52 
Last updated: 2010-11-12 01:59:53 
Author: Ginko  
 
A quick and dirty script to convert youtube's transcripts (xml format) to .srt subtitle files.\n\nTo download youtube's transcript, use this url: http://video.google.com/timedtext?lang=en&v=VIDEO_ID (replace "VIDEO_ID" by the ID which is in the video URL).\n\nYou can easily use this converter in a script which could download the transcript by importing it and then call the main function.
