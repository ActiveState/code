## Monday of first week per year (not first Monday) 
Originally published: 2012-10-31 04:19:30 
Last updated: 2012-10-31 04:37:28 
Author: Scott S-Allen 
 
13th week of any year is ambiguous without identifying the first week. \n\nThis recipe calculates Monday of the first week, not the first Monday, and differs from others by not using flow-control statement (if).
