###Prints full name of all occurrences of given filename in your PATH

Originally published: 2009-06-29 15:20:10
Last updated: 2009-06-29 15:20:10
Author: Ben Hoyt

Simple program to print the full name of all occurrences of the given filename in your PATH. Kind of like the Unix "which" utility, but works for DLLs and other files as well.\n\nUsage: findinpath.py filename
