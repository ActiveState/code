## DateTimeSyncer with Daylight Savings Time adjustment 
Originally published: 2008-10-30 19:33:37 
Last updated: 2008-10-31 15:52:18 
Author: Jack Trainor 
 
DateTimeSyncer synchronizes computer's current date and time with Daylight Savings adjustment\ndirectly without depending on operating system or Python's time.localtime().\n\nThis allows programmers to update Daylight Savings Time rules without\ndepending on external updates to OS or Python.\n\nDouble-clicking this module will synchronize your computer's date and time on a Windows machine. It can be easily extended other time zones and other operating systems.
