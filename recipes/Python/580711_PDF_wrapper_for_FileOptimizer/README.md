## PDF wrapper for FileOptimizer  
Originally published: 2016-10-29 11:47:03  
Last updated: 2016-10-29 20:22:27  
Author: Harald Lieder  
  
Among dozens of other filetypes, FileOptimizer also compresses PDFs - often significantly.\nThe issue is that the used plugin *smpdf* is free for non-commercial use only and it annoyingly **also overwrites metadata** information to state this.\n\nThe following tool remedies these metadata changes (but not the license situation!).
