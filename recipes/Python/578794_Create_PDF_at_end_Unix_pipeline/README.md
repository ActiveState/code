## Create PDF at the end of a Unix pipeline with PDFWriter 
Originally published: 2013-12-22 22:18:59 
Last updated: 2013-12-22 22:19:00 
Author: Vasudev Ram 
 
This recipe shows how to create PDF output at the end of a Unix or Linux pipeline, after all the text processing required, is done by previous components of the pipeline (which can use any of the standard tools of Unix such as sed, grep, awk, etc., as well as custom programs that act as filters).\n
