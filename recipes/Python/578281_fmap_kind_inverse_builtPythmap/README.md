###fmap(): a kind of inverse of the built-in Python map() function

Originally published: 2012-10-06 22:15:16
Last updated: 2012-10-06 22:15:17
Author: Vasudev Ram

The Python map() function returns a list of the results of applying the function to the items of the argument sequence(s).\n\nThe fmap() function does the inverse, in a sense. It returns the result of applying a list of functions to a given argument.\n
