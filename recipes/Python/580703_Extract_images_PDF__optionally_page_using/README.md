###Extract images of a PDF - optionally by page using PyMuPDF / fitz

Last updated: 2016-09-28 12:03:59

Two small scripts to extract images contained in a PDF document as PNG files.\n(1) Script 1 extracts **all** images\n(2) Script 2 extracts only images that are referenced by a page
