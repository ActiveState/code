## Whitespace Interpreter

Originally published: 2010-03-14 15:36:52
Last updated: 2010-05-24 12:02:11
Author: Stephen Chappell

Runs programs in "Programs" and creates *.WSO files when needed.\n\nCan be executed directly by double-click or on the command line.\n\nIf run on command line, add "ASM" flag to dump program assembly.
