###Hashing Nonhashable / Mutable Objects

Originally published: 2015-04-16 12:08:23
Last updated: 2015-04-16 12:11:34
Author: Max Moroz

This class is intended as a wrapper for non-hashable objects (such as lists, dictionaries, etc.) in situations where the object is known to be immutable. Example: memoization of function arguments.\n\nFeedback much appreciated, I will update to reflect any comments.
