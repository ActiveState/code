###Write a PNG image in native Python

Originally published: 2010-10-28 12:03:51
Last updated: 2010-10-28 12:03:52
Author: Campbell Barton

This is a simple PNG writing function, intended to be as small as possible.\nIt only supports RGBA 32bit PNGs and the input should be a buffer of rgba bytes.\n\nTested for python 2 and 3.
