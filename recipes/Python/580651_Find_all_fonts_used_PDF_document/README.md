## Find all fonts used in a PDF document by page

Originally published: 2016-04-25 10:42:46
Last updated: 2016-08-26 00:02:48
Author: Jorj X. McKie

Finds all fonts used in a PDF document by page.\nThis new script is based on PyMuDF v1.9.2 and works for PDF files only.\nHowever, it is a lot simpler, speed has drastically improved and there is no dependency on other packages any more.
