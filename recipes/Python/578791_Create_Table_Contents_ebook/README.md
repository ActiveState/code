## Create a Table of Contents for an ebook with XMLtoPDFBook 
Originally published: 2013-12-16 21:49:32 
Last updated: 2013-12-16 21:49:33 
Author: Vasudev Ram 
 
\nXMLtoPDFBook is a tool that allows you to create simple PDF ebooks from XML content. It is a part of the xtpodf toolkit.\n\nYou can now create PDF ebooks with XMLtoPDFBook, that include a Table of Contents (TOC).\n
