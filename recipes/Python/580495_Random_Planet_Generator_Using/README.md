## Random Planet Generator Using Plasma Fractal 
Originally published: 2016-01-25 23:05:25 
Last updated: 2016-01-25 23:05:25 
Author: FB36  
 
Random Planet Generator using 3D Plasma Fractal and Voxel-based Ray Tracing for rendering.\n\nInstead of generating 2D Plasma Fractal and projecting onto sphere, it generates 3D Plasma Fractal (cube) and cuts sphere from it.
