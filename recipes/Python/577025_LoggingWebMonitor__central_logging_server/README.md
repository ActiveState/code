###LoggingWebMonitor - a central logging server and monitor.

Originally published: 2010-02-02 00:20:13
Last updated: 2010-02-02 01:56:42
Author: Gabriel Genellina

LoggingWebMonitor listens for log records sent from other processes running in the same box or network.  Collects and saves them concurrently in a log file.  Shows a summary web page with the latest N records received.\n
