## Validate XML Files. 
Originally published: 2003-06-26 22:40:30 
Last updated: 2003-06-26 22:40:30 
Author: Patrick Finnegan 
 
This script validates XML files while recursively traversing a specified directory tree.  It requires three packages fileutil, tnc and tdom.  Tnc and tdom are not part of the standard Tcl distro.\n\nTdom is available from http://www.tdom.org/\nTdom tutorial is available from http://mini.net/tcl/1948\n\nTnc is included with tdom zip file.\nTnc info at http://mini.net/tcl/4577 and http://www.tdom.org/tnc.html.
